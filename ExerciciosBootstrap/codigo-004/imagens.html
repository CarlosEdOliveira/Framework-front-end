Image quadrada com borda arredondada

Essa classe de imagem apresenta um formato geométrico representado dentro do código. É representada pela tag: <class=rounded>

Imagem de formato circular

Essa classe de imagem apresenta um formato de bordas arredondadas em um formato geométrico circular. A tag que representa esse tipo de imagem é: <class=rounded circle>

Imagem de quadrada com detalhe

Essa classe de imagem apresenta um formato geométrico quadrado com uma borda quadrada em destaque. A tag que representa esse tipo de imagem é: <class=img-thumbermail>

Imagem fluida

A imagem fluida é uma imagem adaptativa com o padrão das telas. Esse tipo de imagem é expressa com a tag <class=img fluid>
